## 摘要
现有的视觉语言导航（VLN）方法通常采用以自我为中心、逐步进行的范式，这种方法难以避免误差累积且效率受限。虽然近期方法尝试利用预构建的环境地图，但它们往往依赖于增量更新的记忆图或对离散路径提案进行评分，这限制了连续空间推理并造成了离散瓶颈。我们提出俯视VLN（TD-VLN），将导航重新表述为预构建俯视地图上的全局单步路径规划问题，并由我们新构建的R2R-TopDown数据集支持。为解决这一问题，我们引入NavOne——一个统一的框架，该框架通过单次端到端前向传播直接预测多模态地图上的稠密路径概率。NavOne采用俯视地图融合器实现联合多模态地图表示，并扩展了注意力残差以实现空间感知的深度混合。在R2R-TopDown上的大量实验表明，NavOne在基于地图的VLN方法中达到了最先进的性能，其规划阶段速度比现有基于地图的基线快8倍，比以自我为中心的方法快80倍，实现了高效的全局导航。

## Abstract
Existing Vision-Language Navigation (VLN) methods typically adopt an egocentric, step-by-step paradigm, which struggles with error accumulation and limits efficiency. While recent approaches attempt to leverage pre-built environment maps, they often rely on incrementally updating memory graphs or scoring discrete path proposals, which restricts continuous spatial reasoning and creates discrete bottlenecks. We propose Top-Down VLN (TD-VLN), reformulating navigation as a one-step global path planning problem on pre-built top-down maps, supported by our newly constructed R2R-TopDown dataset. To solve this, we introduce NavOne, a unified framework that directly predicts dense path probabilities over multi-modal maps in a single end-to-end forward pass. NavOne features a Top-Down Map Fuser for joint multi-modal map representation, and extends Attention Residuals for spatial-aware depth mixing. Extensive experiments on R2R-TopDown show that NavOne achieves state-of-the-art performance among map-based VLN methods, with a planning-stage speedup of 8x over existing map-based baselines and 80x over egocentric methods, enabling highly efficient global navigation.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 提出**Top-Down VLN (TD-VLN)** 范式：给定语言指令和多模态俯视地图（RGB、占据、语义），直接一步预测完整路径和终点分布，再通过后处理提取可执行轨迹。
- 构建**NavOne**统一框架，实现端到端学习。

### 关键技术细节
1. **Top-Down Map Fuser（俯视地图融合器）**：
   - 将RGB地图（3通道）、占据图（1通道）、语义嵌入图（8通道）按通道拼接，形成12通道输入张量。
2. **Path Former（路径形成器）——编码器-解码器架构**：
   - **多模态融合**：
     - 拼接后的地图被分割成16×16的patch，线性投影为视觉token，并添加位置编码。
     - 语言指令由冻结的BERT编码为文本token。
     - 初始姿态（位置p0和旋转r0）通过线性投影加到视觉token上（**元素级加法**，基于特征正交性假设）。
     - 视觉token作为query，文本token作为key/value，通过12层交叉注意力逐步融合语言信息。
   - **ViT编码器 + 全注意力残差（Full Attention Residuals）**：
     - 12层Transformer编码器，采用Pre-LayerNorm。
     - 引入**全注意力残差**：每层都对所有之前层的输出进行深度加权混合，允许各token从不同表示深度中受益。
     - 进一步提出**空间感知深度查询（Spatial-Aware Depth Query）**：每个token的深度查询由其patch位置嵌入通过零初始化的线性投影调制，实现**位置相关的深度混合**。
   - **解码器**：
     - 轻量级渐进上采样解码器，通过多个转置卷积将编码特征逐步恢复至原始分辨率（256×256），最终输出2通道：路径概率图和目标概率图。
3. **Path Extractor（路径提取器）**：
   - **目标定位**：取目标概率图的最大值点作为预测目标g*。
   - **代价地图构建**：对可通行区域，将路径概率取倒数作为代价；障碍物区域赋予大惩罚成本Cobs=1000。
   - **A*搜索**：在代价地图上采用8-连通和欧氏距离启发式，从起点到目标搜索最优路径。若不可达，则调整到最近可通行单元格。
4. **损失函数**：
   - 多任务目标：路径BCE损失 + 目标BCE损失（权重α=5） + 连续性正则化（梯度平滑、起始点锚定、形态学侵蚀损失，总权重λ=0.1）。

## 7. 优点：方法或实验设计上的亮点

- **范式创新**：将VLN重新定义为全局单步规划，克服了逐步骤范式的累积误差和效率瓶颈。
- **端到端学习**：直接从原始地图和语言预测连续路径概率，无需离散提案或增量记忆，实现连续空间推理。
- **高效推理**：单次前向传播即可生成完整路径，速度优势显著。
- **充分实验**：包含架构消融、模态消融、鲁棒性分析、特征正交性验证、跨真实机器人地图泛化，证明方法的全面性和实用性。
- **空间感知深度混合**：提出位置相关的全注意力残差，是一种新颖且有效的特征融合机制。

## 8. 不足与局限
