## 摘要
具身指令跟随（EIF）要求智能体在交互式3D环境中理解并执行复杂的自然语言指令。尽管近期取得了进展，现有方法在处理长程规划和不可逆状态变化时常常失败，导致任务成功率低。为了解决这些挑战，我们提出了RePlan-Bot，一种新型EIF智能体，它在任务执行过程中进行多级连续重规划。RePlan-Bot集成了一个基于LLM的高层级审计器，根据环境反馈动态调整子目标；一个基于多层实例图的常识引导搜索机制，用于精确和结构化的物体定位；以及一个基于ViT的轻量级校正器，用于预先修复有风险的底层动作。在ALFRED基准测试上评估，RePlan-Bot在可见和未见环境中均达到了最先进的性能，展示了优越的适应性和可靠性。

## Abstract
Embodied instruction following (EIF) requires agents to understand and execute complex natural language commands within interactive 3D environments. Despite recent advances, existing methods often fail in long-horizon planning and handling irreversible state changes, resulting in low task success rates. To address these challenges, we introduce RePlan-Bot, a novel EIF agent that performs multi-level, continuous replanning throughout task execution. RePlan-Bot integrates a high-level LLM-based auditor for dynamic sub-goal adjustments guided by environmental feedback, a commonsense-guided search mechanism based on a multi-layered instance map for precise and structured object localization, and a lightweight ViT-based corrector to preemptively fix risky low-level actions. Evaluated on the ALFRED benchmark, RePlan-Bot achieves state-of-the-art performance in both seen and unseen environments, demonstrating superior adaptability and reliability.

## 2. 方法论：核心思想、关键技术细节

### 核心思想
将EIF分解为三个层级，各自集成重规划能力，形成闭环：高层调整子目标序列、中层指导物体搜索、低层修正执行动作。

### 关键技术细节

- **高层重规划：环境感知的LLM审计器**
  - 当任务开始时或代理花费过多步骤仍未完成子目标时触发。
  - 审计器接收结构化提示（系统角色、指令、环境反馈中的已观察物体），输出修正后的子目标列表。
  - 两种具体作用：
    1. **执行前子目标修正**：修正如缺少“打开柜子”等前提条件，或纠正对象引用（如“Cup”调整为“Mug”）。
    2. **基于视觉的物体消歧**：根据检测到的物体集合替换原计划中不可见或错误的目标。

- **中层搜索：常识引导搜索机制**
  - 构建**多层实例图**：将3D观测按垂直高度切片，每个像素标记语义类别和实例ID，并随时间用IoU跟踪实例。
  - 当目标物体未观测到时，LLM根据任务与当前检测物体预测一个**宿主类别**（例如“海绵常在柜子里”）。
  - 智能体通过查询多层实例图，依次访问所有宿主类别实例（如逐个检查多个柜子），直到发现目标。
  - 避免依赖U-Net输出的粗糙热图。

- **低层重规划：视觉驱动的动作修正器（ViT-based Corrector）**
  - 在每个原始低层动作执行前，ViT模块以当前RGB图像和计划动作作为输入，输出修正后的动作（如MoveAhead、RotateLeft等）。
  - 训练样本来自执行轨迹：将RGB图像和原始动作编码后输入Transformer，用softmax分类头预测正确动作。
  - 当修正动作与原始动作不同时，执行修正动作，从而预防因距离过远、视角过低或被遮挡导致的失败。
