## 摘要
视觉-语言导航（VLN）方法目前遵循两种主要范式：一种是在导航轨迹上微调端到端视觉-语言模型（VLM）策略以直接预测动作，另一种是集成预训练多模态大语言模型（MLLM）的零样本模块化流水线，以实现对未见环境的无训练泛化。然而，端到端方法难以应对长程导航且缺乏动态推理能力，而零样本方法则受限于空间定位不足，无法进行可靠规划，且需要大量推理时间。为弥合这一差距，我们提出了SEDualVLN，一种空间增强的双系统VLN框架。系统1是一个增强全局和局部空间感知的VLM模型，用于动作生成。系统2集成通用MLLM与地图模块，其中MLLM通过利用实时3D地图的俯视图以及渲染路径图像流来规划路标点。两个系统利用不同形式的空间增强，以培养VLN任务中智能体的方向感。最终，它们通过快慢协同方式合作完成导航任务。SEDualVLN在VLN-CE基准上实现了最先进的性能，进一步的消融研究证明了每个系统和模块的有效性。

## Abstract
Vision-Language Navigation (VLN) approaches have currently followed two primary paradigms: the end-to-end Vision-Language Model (VLM) policy fine-tuned on navigation trajectories to directly predict actions, and the zero-shot modular pipeline integrating pre-trained Multimodal Large Language Model (MLLM) for training-free generalization to unseen environments. However, end-to-end methods struggle with long-horizon navigation and lack dynamic reasoning, whereas zero-shot methods are constrained by limited spatial grounding for reliable planning and also require substantial reasoning time. To bridge this gap, we introduce SEDualVLN, a spatially-enhanced dual-system VLN framework. System 1 is a VLM model enhanced with both global and local spatial awareness, used for action generation. System 2 integrates a general MLLM with a mapping module, wherein the MLLM plans waypoints by leveraging top-down views of the real-time 3D map alongside streams of rendered path images. Both systems leverage different forms of spatial enhancement to cultivate the agent's sense of direction in VLN tasks. Ultimately, they cooperate to complete the navigation task through a fast-slow coordinated approach. SEDualVLN achieves state-of-the-art performance on VLN-CE benchmarks, and further ablation studies demonstrate the effectiveness of each system and module.

# SEDualVLN 论文详细总结

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：视觉-语言导航（VLN）要求智能体在未知环境中依据自然语言指令进行连续导航。现有方法分为两类：
  - **端到端 VLM 方法**：通过微调统一策略直接预测动作，但在长程导航中累积误差大、缺乏动态逐步推理能力。
  - **零样本 MLLM 方法**：利用预训练多模态大模型进行零样本规划，泛化性好，但空间基础（spatial grounding）不足，几何推理弱，且推理耗时高。
- **核心挑战**：如何同时提升端到端模型的空间感知和零样本模型的空间推理可靠性，使智能体具备鲁棒的方向感。
- **本文目标**：设计一个 **双系统框架**，结合两种范式的优势，并通过显式空间增强提升导航成功率。

## 2. 方法论：核心思想、关键技术细节

### 2.1 整体架构：双系统快慢协同
- **System 1（快系统）**：基于 RGB 视频流的 VLM 模型，负责生成低层动作（前进、左转、右转、停止），快速执行。
- **System 2（慢系统）**：集成通用 MLLM（如 GPT-4o）与 3D 地图模块，负责规划路点（waypoint），提供全局指导。
- **调度机制**：System 1 以高频运行（每步生成动作），System 2 低频规划（如每 20 步触发一次），两者通过快慢协调循环合作。

### 2.2 System 1：空间增强的 VLM（基于 StreamVLN 改进）
- **基础模型**：LLaVA-Video-7B（语言骨干 Qwen2-7B），采用多轮对话、KV Cache 和空间 token 剪枝。
- **全局空间增强策略（Global Spatial Enhancement）**：
  - 从预训练 3D 基础模型 VGGT 提取空间特征，作为隐式监督信号。
  - 在 VLM 中间注意力层（第 24 层最佳）对视觉 token 投影后的特征与 VGGT 特征计算余弦距离，构成空间对齐损失。
  - 损失函数：`L_SE = L_action + α * (1/N) * Σ [1 - cos(V_t, S_t + p_t)]`，其中 α 平衡动作损失与空间损失。
- **局部空间增强策略（Local Spatial Enhancement）**：
  - 针对通道/门等连接结构，使用 Grounding DINO 检测通道区域，SAM 分割得到二值掩码，再通过 MLP 投影为与全局视觉 token 同维的局部 token，输入 VLM 增强对通道感知。

### 2.3 System 2：映射–渲染–推理 MLLM（三阶段流水线）
- **Mapping 阶段**：利用实时 3D 地图（基于 LingBot-Map）和 2D 前沿地图（frontier map），增量式构建环境地图，指导探索未访问区域。
- **Rendering 阶段**：
  - 对每个前沿点（frontier），用 A* 算法计算无碰撞路径。
  - 沿路径插值虚拟相机位姿并渲染 RGB 图像，然后使用 CLIP 编码进行余弦相似度剪枝，去除冗余帧。
- **Reasoning 阶段**（两阶段）：
  - **阶段1**：向 GPT-4o 提供 3D 俯视图，要求描述智能体当前位置、环境结构、物体关系及可能方向。
  - **阶段2**：提供各前沿路径的渲染图像序列，让 GPT-4o 比较路径与指令的匹配度，选出最优前沿点作为路点。

## 6. 主要结论与发现

- **性能领先**：SEDualVLN 在 R2R-CE Val-Unseen 上 SR 达 67.3%（比 DualVLN 高 3%），在 RxR-CE 上 SR 达 63.9%（高 2.5%），刷新 SOTA。
- **空间增强有效性**：全局空间增强在 VLM 第 24 层监督效果最佳，SR 提升 6.0%（R2R）和 14.4%（RxR）；局部空间增强 SR 提升 3.5%（R2R）和 4.1%（RxR）。
- **双系统协作优势**：相比单独使用 System 1 或 System 2，双系统协作在 SR 上分别提升 4.5% 和 7.5%，且平均完成时间比纯 System 2 降低约 5~6 倍；最佳频率比为 20:1。
- **GPT-4o 表现最优**：在零样本路点规划中，GPT-4o 的 SR 略高于 Gemini 和 Qwen2.5-VL。

## 7. 优点

- **视角新颖**：将双系统思想与显式空间增强深度结合，而非简单组合两个现有范式。
- **方法设计巧妙**：
  - 隐式监督 VLM 中间层，避免依赖传感器深度，仅用 RGB 即可学习空间语义。
  - 局部通道提取模块专门针对走廊/门等关键决策点，提升 VLM 在岔路口的判断。
  - System 2 的“映射-渲染-推理”流水线为 MLLM 提供了物理一致的 3D 上下文和动态视觉线索。
- **实验扎实**：消融实验覆盖多个设计维度，验证了每个组件的贡献。
