## 摘要
现有的目标导航基准通常告诉具身代理需要找到哪类物体，例如微波炉或椅子。面向人类的具身AI经常被问到不那么直接的问题：“我需要加热这个食物”或“房间感觉闷”。代理必须推断出能满足需求的物体，找到一个场景中的实例，并决定是否已达到目标。我们将此场景作为意图驱动的目标导航进行研究，并引入IntentionNav，这是一个从隐式人类指令进行主动目标搜索的诊断基准。每个回合提供一个自由文本的意图、RGB-D观测和位姿，但不提供目标物体名称。IntentionNav包含176个Isaac Sim场景和64个目标类别上的500个意图。每个意图以四种受控的指令风格重写，并用四种意图模式之一进行注释，在匹配几何条件下将表面措辞与语义提示类型分开。这种配对设计支持对目标推理、语言鲁棒性、邻接可达性和终端成功（而非仅总体成功）的分析。我们使用一个固定的主动导航代理评估了三个视觉语言模型。模型在48.3%的回合中识别出预期目标，在68.7%的回合中进入其2米邻域，但只有24.9%的回合成功终止，在5.5%的回合中实现了基于地面的1米成功。事件脚本意图的成功率最高（28.7%），而物理状态和可操作意图的成功率较低（分别为19.2%和18.5%），这表明间接的人类意图仍然是主动具身搜索中目标选择、视觉验证和终端定位的瓶颈。

## Abstract
Existing object navigation benchmarks usually tell an embodied agent which object category to find, such as microwave or chair. Human-facing embodied AI is often asked something less direct: "I need something to warm this food" or "the room feels stuffy." The agent must infer the object that can satisfy the need, find a scene-grounded instance, and decide whether the goal has been reached. We study this setting as intent-driven object navigation and introduce IntentionNav, a diagnostic benchmark for active object search from implicit human instructions. Each episode provides a free-text intent, RGB-D observations, and pose, but withholds the target object name. IntentionNav contains 500 intents over 176 Isaac Sim scenes and 64 target categories. Each intent is rewritten in four controlled instruction styles and annotated with one of four intent modes, separating surface phrasing from semantic cue type under matched geometry. This paired design supports analysis of target inference, language robustness, neighborhood reachability, and terminal success rather than only aggregate success. We evaluated three VLMs using a fixed active-navigation agent. Models identify the intended target in 48.3 percent of episodes and enter its 2 m neighborhood in 68.7 percent, but terminate successfully in only 24.9 percent and achieve grounded 1 m success in 5.5 percent. Success is highest for event-script intents (28.7 percent) and lower for physical-state and affordance intents (19.2 percent and 18.5 percent), showing that indirect human intent remains a bottleneck for target selection, visual verification, and terminal localization in active embodied search.

- **问题**：现有目标导航（ObjectNav）基准通常要求智能体依据显式类别标签（如“微波炉”）寻找物体；但人类在真实交互中常以**隐含意图**（如“我需要加热这个食物”）表达需求，智能体需要自行推断目标物体。
- **意义**：当前缺乏一个能系统评估智能体从间接人类意图中恢复目标、通过主动探索定位物体并确认成功的诊断基准。现有工作（DDN、ULN）要么只关注可操作性需求、要么聚焦于路径跟随而非物体搜索，且缺乏对语言形式与语义提示类型的正交分解。
- **整体目标**：通过构建 **IntentionNav** 基准，隔离意图理解、邻接可达、终端定位和视觉验证等环节的失败模式，揭示隐含意图驱动的具身搜索瓶颈。

## 2. 提出的方法论：核心思想、关键技术细节与算法流程
- **核心思想**：设计一个**诊断性基准**，对每个意图同时标注“指令风格”（正式/自然/随意/情感）和“意图模式”（事件脚本/内在状态/物理状态/可操作性），在**相同场景、相同目标、相同起始位姿**下比较不同语言条件对导航成功率的影响，从而分离措辭效应与语义推理难度。

  - **任务形式化**：每个回合 e = (场景 s, 自由文本意图 u, 起始位姿 q₀, 隐藏目标 o⋆, 类别 c⋆, 位置 p⋆)，智能体只能通过 RGB-D 和位姿探索，不能接收类别标签。

  - **诊断语言轴**：

## 6. 主要结论与发现
- **整体表现**：三个模型在意图匹配（IM=0.483）和邻接可达（OSR=0.687）上表现尚可，但终端成功急剧下降（SR=0.249），地面验证成功仅 **0.055**。
- **关键瓶颈**：
  - **邻接到终端间隙（gₜₑᵣₘ=0.438）**最大，说明智能体常在到达目标邻域后因视觉证据不足或决策失误而失败。
  - 意图间隙（gᵢₙfₑᵣ=0.234）次之，表明目标推理仍具挑战。
  - 精度间隙（gₚᵣₑc=0.194）显示近距离定位失败普遍。
- **意图模式差异**：事件脚本意图成功率最高（28.7%），物理状态（19.2%）和可操作性（18.5%）最低，说明需要“环境状态推理”和“纯功能推断”的任务更困难。
- **指令风格**：总体影响较小（Δstyle=0.031），但跨风格一致性 CSR 仅 5.6%-9.6%，表明单个意图在不同措辞下表现不稳定。
- **显式目标消融**：将意图替换为类别标签后 SR 从 28.0% 提升至 44.8%，但仍有一半以上失败，说明视觉验证和终端定位是独立瓶颈。

## 7. 优点
1. **诊断性设计**：通过正交语言轴和链式指标（IM、OSR、SR、GSR）分解失败阶段，而非仅汇报单一成功率。
2. **严格的控制变量**：每个意图的四种风格共享相同场景、目标和起始位姿，可纯净测量措辭效应。
3. **高质量基准**：经过自动生成+人工审核+词法泄漏检查、物理唯一性过滤、多轮 VLM 评分，确保指令可回答且无直接目标泄露。
4. **开源与可复现**：提供代码、同义词表、提示模板、评估脚本；明确报告算力和依赖。
5. **公平比较**：所有模型使用完全相同的导航代理和超参数，仅 VLM 不同，避免方法工程的混淆。

## 8. 不足与局限
1. **语言与场景限制**：仅限于英语单目标指令、静态公寓场景（无动态物体、无多人交互）、无对话澄清机制。
2. **任务范围窄**：单目标搜索，无多目标、时序依赖或开放世界操作。
3. **模拟与现实鸿沟**：在 Isaac Sim 模拟器中评估，未在真实机器人上验证；模拟中的感知和物理与现实差距可能存在。
4. **评估指标局限性**：意图匹配依赖有限同义词表，可能遗漏合理同义表达；自动 VLM 评分器可能存在偏见。
5. **参考代理固定**：当前结论仅基于一种固定导航策略，不同导航算法可能产生不同失败模式。
6. **时间预算固定**：30 步限制可能不适合所有场景（论文提及 40 步检查未改变排序，但深层影响未全面探索）。
7. **文化多样性未覆盖**：意图表达方式可能因文化而异，基准未包含非西方语境。
