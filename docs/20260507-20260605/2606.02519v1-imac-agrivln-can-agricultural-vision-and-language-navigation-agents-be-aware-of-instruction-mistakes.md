## 摘要
农业机器人正在作为强有力的助手执行各种农业任务，然而，它们仍然严重依赖人工操作或轨道系统进行移动。AgriVLN方法和A2A基准开创性地将视觉与语言导航（VLN）扩展到了农业领域，使机器人能够按照自然语言指令导航到目标位置。然而，几乎所有先前的方法都采用了一个理想假设，即给定的指令本身是正确的，这与现实场景不符，因为任何人在下达指令时都可能出现错误。为了填补这一空白，我们提出了A2A-MI基准，在其中构建了一个半自动数据标注器，以更多样化和高效的方式将三种错误分类插入到每个原始指令中。我们测试了几种最先进的农业VLN智能体，观察到其在SR上下降了57%、在NE上下降了9%，这表明农业VLN智能体倾向于假设给定指令是正确的，因此当它看到的场景与接收到的指令不一致时，没有怀疑指令的意识。为了建立对指令错误的意识，我们提出了IMAC模块，该模块分析指令和当前正面图像，判断指令是否有错误，并在必要时尝试纠正。我们将IMAC集成到基线模型中，观察到了显著的改进，充分缩小了与无错误指令性能之间的差距。项目地址：https://github.com/AlexTraveling/IMAC-AgriVLN。

## Abstract
Agricultural robots are serving as powerful assistants across a wide range of agricultural tasks, nevertheless, still heavily relying on manual operations or railway systems for movement. The AgriVLN method and the A2A benchmark pioneeringly extended Vision-and-Language Navigation (VLN) to the agricultural domain, enabling a robot to navigate to a target position following a natural language instruction. However, almost all the prior methods adopt an ideal assumption that the given instructions themselves are correct, which does not align with the realistic scenarios, because anybody may say an instruction with mistakes. To bridge this gap, we propose the A2A-MI benchmark, in which we build a semi-automatic data annotator to insert three mistake classifications into each original instruction in a more diversified and efficient way. We test several state-of-the-art agricultural VLN agents on it and observe a sufficient drop with -57% on SR and -9% on NE, from which we suggest that an agricultural VLN agent tends to assume that the given instruction is correct, so does not have the awareness to doubt it when the scenes it sees do not align with the instruction it receives. To build the awareness on instruction mistake, we propose the IMAC module analyzing the instruction and the current front-facing image, to judge whether the instruction has mistakes and attempt to correct it when needed. We integrate IMAC into the baseline model, and observe a noteworthy improvement, sufficiently narrowing the gap to the performance on instructions without mistakes. ### 2. 方法论：核心思想、关键技术细节

- **核心思想**：提出IMAC（Instruction Mistake Awareness and Correction）模块，通过分析指令与当前正面相机图像，判断指令是否存在错误，并在必要时尝试纠正。IMAC集成到AgriVLN基线中，使智能体具备对指令错误的感知与纠正能力。
- **关键技术细节**：
  - **IMAC模块**：
    - 输入：当前指令 \(W\) 和正面图像 \(I_t\)。
    - 利用视觉语言模型（VLM） \(AC(\cdot)\) 分析两者，输出：错误区间 \(\hat{S}\)、纠正后指令 \(\hat{W}\)、决策摘要 \(\rho_t\)。
    - 遵循两个原则：
      - **保守验证**：仅当高度确信指令错误且可能导致导航失败或不安全行为时才标记错误，避免过度敏感。
      - **最小干预**：只纠正最小的错误词或区间，不做不必要的风格改进。
    - 若 \(\hat{S} = \text{“None”}\)，则不操作；否则，替换错误区间。
  - **集成到AgriVLN基线**：
    - 首先在起始时刻（\(t=0\)）对整条指令应用IMAC，得到处理后的指令 \(\hat{W}\)。
    - 用大语言模型（LLM） \(S(\cdot)\) 将 \(\hat{W}\) 分解为有序子任务列表（Subtask List）。
    - 在每个时间步 \(t\)，再次应用IMAC（输入当前执行子任务 \(D_{i^*}\) 和图像 \(I_t\)），若发现错误则更新该子任务的描述、起止条件。
    - 然后由VLM决策器 \(D(\cdot)\) 基于当前子任务和图像输出低层动作（[FORWARD], [LEFT ROTATE], [RIGHT ROTATE], [STOP]）。
    - 终止条件：预测到[STOP]、动作序列与真值偏离超过4秒、或达到最大时间步。
- **公式/算法流程**（文字说明）：
  1. 输入指令 \(W\)，利用IMAC得到纠正后指令 \(\hat{W}\)。
  2. 利用LLM将 \(\hat{W}\) 分解为有序子任务列表。
  3. 每步 \(t\)，IMAC再次检查当前子任务与图像，必要时更新子任务。
  4. VLM决策器根据当前子任务和图像选择动作，更新子任务状态。
  5. 重复直至满足终止条件。

### 3. 实验设计：数据集、基准、对比方法

- **数据集与基准**：
  - 提出 **A2A-MI** 基准，基于原A2A基准，包含1,560个episode，覆盖6种场景（农场、温室、森林、山地、花园、乡村）。
  - 半自动数据标注器：先用LLM为每个指令生成三类错误候选（描述性形容词、具体名词、行为动词），再由人工专家从每类中选择最合适的一个。最终A2A-MI包含4,680条指令（仅评估），平均长度46词。
- **对比方法**：
  - 基线方法：**SIA-VLN**、**DILLM-VLN**、**AgriVLN**（本文基线）。
  - 上下界：**Random**（随机动作）、**Fixed**（固定动作）、**Human**（人类表现）。
  - 本文方法：**IMAC-AgriVLN**（基线+IMAC模块）。
- **评估指标**：
  - **Success Rate (SR)**：成功到达目标（2米内）的比例。
  - **Navigation Error (NE)**：停止位置与目标位置的距离（米）。
  - **Instruction Mistake Valid Awareness Rate (AR)**：针对每个episode，若存在某时刻IMAC预测的错误区间与标注一致，则AR=1，否则为0。平均后得到整体AR。

### 6. 主要结论与发现

- **核心结论**：现有农业VLN智能体（如AgriVLN）假设指令正确，缺乏对指令错误的感知能力，导致在A2A-MI上SR下降57%、NE上升9%。
- **IMAC效果**：集成IMAC后，SR从0.10提升至0.14，NE从4.81m降至4.79m，AR平均为0.27（即能识别超过四分之一的错误）。显著缩小了与无错误指令性能的差距，但尚未完全恢复。
- **错误类别影响**：动词错误对性能影响最大（最误导），名词错误相对较轻；语义权重高的行为动词是双刃剑。
- **IMAC与STL互补**：两者同时使用效果最佳，IMAC主要负责错误感知与纠正，STL提供子任务分解辅助。
- **失败案例揭示**：IMAC可能过度激进，将正确词误判为错误（如将“person”改为“cornfield”），导致后续导航恶化；保守策略更可取。
