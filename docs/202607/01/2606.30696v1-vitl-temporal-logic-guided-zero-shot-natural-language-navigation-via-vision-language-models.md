## 摘要
使机器人能够遵循自然语言命令完成零样本长时任务仍然具有挑战性。它需要从自然语言命令中提取隐含的时间约束和逻辑约束，并相应地执行多个子任务。最近的零样本目标导航方法使用视觉语言模型（VLM）引导未知环境中基于前沿的探索，但仅限于单目标任务。现实世界中的命令，如“清理椅子或沙发，然后打开电视”，需要在时间约束顺序下导航到多个目标，现有的零样本系统无法处理。我们提出ViTL框架，在两个层面解决这一差距。在任务层面，我们使用大型语言模型（LLM）将自然语言命令编译成线性时间逻辑（LTL）公式，然后转换为确定性有限自动机（DFA），协调多通道价值图并在检测到新物体时触发动态重规划。在导航层面，我们引入方向得分：不是在整个视野中产生与方向无关的价值，而是在观察图像上标记前沿方向，并从VLM中提取每个方向得分。在Habitat-Matterport 3D（HM3D）上的实验表明，完整框架实现了具有时间约束的零样本长时自然语言导航任务完成，并且方向得分提高了单目标导航的准确性和效率，优于基线。

## Abstract
Enabling robots to follow natural language commands to complete zero-shot long-horizon tasks remains challenging. It requires extracting implicit temporal and logical constraints from natural language commands and executing multiple sub-tasks accordingly. Recent zero-shot object navigation methods use vision-language models (VLMs) to guide frontier-based exploration in unknown environments, but they are limited to single-target tasks. Real-world commands such as "Clean either the chair or the couch, then turn on the tv." require navigating to multiple targets in a temporally constrained order, which no existing zero-shot system can handle. We present ViTL, a framework that addresses this gap at two levels. At the task level, we use a large language model (LLM) to compile natural language commands into Linear Temporal Logic (LTL) formulas, which are then converted into Deterministic Finite Automata~(DFA) that coordinate multi-channel value maps and trigger dynamic replanning when new objects are detected. At the navigation level, we introduce directional score: rather than producing a direction-agnostic value across the entire field of view, we label frontier directions on the observation image and extract per-direction scores from the VLM. Experiments on Habitat-Matterport 3D (HM3D) show that the full framework enables zero-shot long-horizon completion of natural language navigation tasks with temporal constraints, and that directional score improves single-target navigation accuracy and efficiency over the baseline.

## 2. 提出的方法论

### 2.1 核心思想
- **任务层**：利用 LLM 将自然语言指令一次性编译为 LTL 公式，再转换为确定性有限自动机（DFA）。DFA 的图结构天然编码了所有合法执行顺序，并通过动态边权重实现实时重规划。
- **导航层**：针对单目标子任务，提出**方向性分数（directional score）**——在观测图像上标记各前沿方向，让 VLM 分别评分，从而生成具有空间分辨率的导航价值图，取代传统的一致分配方法。

### 2.2 关键技术细节

#### 任务级规划器
1. **自然语言 → LTL → DFA**：调用 LLM（单次离线查询）提取目标类别集合和时序逻辑，生成形如 `♢((chair ∨ couch) ∧ ♢ tv)` 的 LTL 公式，再用标准工具（如 lomap）编译为 DFA（图2）。
2. **动态目标选择与重规划**：DFA 的每个边关联权重 `w(e)`，基于目标是否已被检测到（`w_det << 1`）、是否已失败（`w_fail >> 1`）或未发现（`1.0`）。每步计算从当前 DFA 状态到接受态的最短加权路径，选择第一条边对应的目标作为当前导航目标。当检测到新物体或目标被标记失败时，边权重更新并重规划。
3. **多通道价值图**：为每个目标类别并行维护一个价值图通道，切换目标时不丢失已有探索信息。

#### 导航级执行器
1. **方向性分数计算**：
   - 将前沿点投影到 RGB 图像上，用字母标签标记（图1）。

### 2.3 算法流程（Algorithm 1）
- 初始化 DFA、目标集、多通道价值图。
- 循环：每步检测新物体 → 更新 DFA 边权重并重规划 → 更新所有价值图通道 → 若当前目标已被检测到则直接导航至其位置，否则根据对应价值图选择前沿探索 → 到达目标后推进 DFA 状态 → 若失败则标记失败并重规划，直至到达接受态或超时。

## 6. 主要结论与发现

1. **单目标导航**：ViTL 的方向性分数方法在 SR（54.1%）和 SPL（30.9）上达到与领先方法相当的水平（VLFM 52.4/30.8，WMNav Gemini 1.5 Pro 58.1/31.2），使用开源 LLaVA-34B 而不依赖闭源大模型。
2. **任务规划**：ViTL 的 LTL → DFA 方法在 2000 个任务中达到 95.3% 的翻译正确率，在此基础上任务完成率 100%（考虑翻译错误则为 95.3%），远优于 LLM-planner 的 88.3%，且探索次数更少。
3. **全系统**：在相互完成的 episodes 上，ViTL 平均路径长度 25.45 m vs. LLM-planner 30.05 m，减少 15.31%，证明了规划优势能转化为实际导航效率提升。
4. **失效模式**：LLM-planner 容易发生违反时序约束（如提前到达 tv）或过早声明任务完成，ViTL 通过 DFA 形式化保证避免了这两类问题。

## 7. 优点

- **创新性**：首次将 LTL/DFA 应用于零样本导航任务规划，形式化保证了时序约束满足。
- **效率**：仅需一次 LLM 离线翻译，后续通过图搜索在线决策，避免每次决策调用 LLM，降低延迟和累积错误。
- **方向性分数**：简单有效的视觉空间推理改进，使 VLM 能区分图像不同区域，提高导航精度。
- **实验设计**：任务规划器部分用概率模型隔离导航噪声，独立评估规划质量，设计巧妙。
- **开放源码与可复现**：使用开源模型和数据集，方法透明。

## 8. 不足与局限

- **导航层假设过度简化**：假设每个子任务只需“到达”物体，忽略动词背后的具体交互（如“清洁”“打开”），限制了在真实机器人上的直接应用。
- **NL→LTL 翻译存在误差**：95.3% 准确率仍会引入 4.7% 的错误公式，且未对错误模式进行分析或提出容错机制。
- **全系统实验规模小**：仅 2 个场景、50 个 episodes，场景选择偏向高成功率，可能高估实际性能。缺少在随机场景或低成功率场景下的全系统评估。
- **缺乏消融实验**：未单独验证方向性分数、时序融合、空间平滑、多通道价值图或动态权重策略的独立贡献。
- **可扩展性**：LTL 表达能力有限，对于更复杂约束（如“到达 A 并保持距离 B 不超过 2m 之后再到达 C”）无法直接处理。

- **潜在偏差风险**：VLM 的场景相关性评分可能受训练数据分布影响，对未见过的物体或布局产生偏见；YOLOv7-E6E 检测器在 HM3D 上的泛化能力未单独评估。
