## 摘要
连续环境中的视觉与语言导航（VLN）要求智能体在以自我为中心的观察中理解指令，同时跨长动作序列保持空间理解。最近的导航基础模型通过扩展视觉-语言模型取得了显著进展，但它们通常将导航主要学习为直接动作生成，而未显式建模世界状态或预测其未来演化。我们提出FutureNav，一种基于VLM的统一世界-动作建模框架，用于视觉与语言导航。具体而言，FutureNav联合编码文本、视觉和空间特征，将其输入LLM，并优化四个目标以实现世界与动作的同步建模：用于导航动作预测的动作策略目标、用于建模状态转移的逆向和正向动力学目标，以及用于预测未来空间状态的未来生成目标。这种统一架构在不牺牲推理速度的前提下，强化了动作预测并显式建模了世界。大量实验表明，仅使用4B规模的主干网络，FutureNav在多个VLN基准测试中达到了最先进性能，并显著优于先前的VLN方法，为未来的VLN世界-动作模型铺平了道路。我们将发布代码和模型以支持未来研究。

## Abstract
Vision-and-language navigation (VLN) in continuous environments requires an agent to ground instructions in egocentric observations while maintaining spatial understanding across long action sequences. Recent navigation foundation models have shown strong progress by scaling vision-language models, but they often learn navigation primarily as direct action generation, without explicitly modeling world states or predicting their future evolution. We introduce FutureNav, a VLM-based unified world-action modeling framework for vision-and-language navigation. Specifically, FutureNav jointly encodes text, visual, and spatial features and feeds them into the LLM, and optimizes four objectives for simultaneous world and action modeling: an action policy objective for navigation action prediction, inverse and forward dynamics objectives for modeling state transitions, and a future generation objective for predicting future spatial states. This unified architecture strengthens action prediction while explicitly modeling the world, without sacrificing inference speed. Extensive experiments show that, with only a 4B-scale backbone, FutureNav achieves state-of-the-art performance on multiple VLN benchmarks and substantially outperforms prior VLN methods, paving the way toward future world-action models for VLN. We will release the code and models to support future research.

- **研究背景**：早期VLN方法依赖显式的导航结构（如航点预测、拓扑规划、语义地图等）；近期VLM方法（NaVid、Uni-NaVid、NaVILA、JanusVLN等）通过大规模预训练实现了更强性能，但仍以直接动作预测为主，缺少对世界状态的显式监督。另一方面，世界模型方法（NWM、WorldVLN、WAM-Nav等）尝试预测未来观测，但往往将策略学习、动力学建模和未来生成作为分离组件，推理时需额外计算成本。**FutureNav旨在填补这一空白**，提出将动作预测与世界建模统一在同一VLM框架中，在不增加推理开销的前提下提升导航性能。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：FutureNav是一个基于VLM的统一世界-动作建模框架，通过联合学习**动作策略、逆向动力学、正向动力学和未来状态生成**四个互补目标，使模型既能预测动作，又能理解动作引起的状态转移和未来空间演化。

- **关键技术细节**：
  1. **输入编码**：
     - **文本与视觉编码**：使用标准VLM（Qwen3-VL）的tokenizer和视觉编码器处理指令和观测帧。
     - **空间编码**：使用**冻结的VGGT**作为空间编码器，从同一帧输出空间感知特征。通过可训练的merger（两层MLP）将空间特征投影到VLM隐空间，并与VLM视觉令牌进行**残差加权融合**（权重α=0.2），形成最终的视觉输入。
  2. **世界-动作建模的四个目标**：
     - **动作策略（Action Policy）**：标准VLM解码界面，根据指令和增强后的视觉令牌，自回归预测导航动作（STOP、MOVE_FORWARD、TURN_LEFT、TURN_RIGHT），损失为交叉熵 L_poicy。
     - **逆向动力学（Inverse Dynamics）**：从相邻两帧观测的变化推断中间动作。将LLM上下文隐状态与当前帧和上一帧的视觉令牌均值池化及其差值拼接，通过分类头预测动作，损失为交叉熵 L_inverse。
     - **正向动力学（Forward Dynamics）**：在教师强制下，将当前观测隐状态与真实动作令牌隐状态拼接，通过MLP回归下一帧的池化空间特征，损失为MSE L_forward。
     - **未来生成（Future Generation）**：不依赖动作令牌，仅将上下文隐状态和当前视觉隐状态拼接，预测下一帧的池化空间特征，损失为MSE L_gen。
  3. **训练目标**：总损失为四项损失的加权和（L = L_poicy + λ_f L_forward + λ_i L_inverse + λ_g L_gen，权重均设为0.1，策略损失权重1.0）。训练时仅优化LLM、merger和四个预测头，视觉编码器和空间编码器冻结。
  4. **推理**：默认只使用动作策略分支进行直接动作解码，无需任何未来预测或候选遍历，因此不增加额外计算成本。

- **公式/算法流程（文字说明）**：
  1. 输入指令x和观测历史O_t。
  2. 经VLM视觉编码器得到视觉嵌入E_q^t，经冻结空间编码器得到空间特征G_t，经merger得到E_g^t，残差融合得E_v^t = E_q^t + α E_g^t。
  3. 将指令文本令牌、视觉令牌送入LLM得到隐状态H_t。
  4. 从H_t中提取相应隐状态，分别计算四个损失并反向传播。
  5. 推理时仅从LLM输出直接解码动作令牌。
