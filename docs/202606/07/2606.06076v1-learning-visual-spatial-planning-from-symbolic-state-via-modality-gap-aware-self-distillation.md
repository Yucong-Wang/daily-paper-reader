## 摘要
尽管视觉-语言模型在通用多模态理解方面表现出色，但在视觉空间规划上仍存在困难。我们将其归因于感知-推理模态差距：视觉规划要求模型从像素中推断潜在状态结构，然后基于恢复的结构进行推理以生成有效动作，而符号规划直接利用显式对象和约束。这造成了视觉状态恢复和多步规划的双重瓶颈。为解决此问题，我们提出了MGSD，一种两阶段模态差距感知自蒸馏框架。首先，冷启动基础阶段为视觉学生模型提供可靠的状态表示，最小化早期感知噪声。其次，特权教师通过在线策略蒸馏传递规划能力，使用显式符号状态来监督学生模型自身的视觉展开前缀。关键在于，符号数据仅在训练期间使用，推理时完全基于视觉。在视觉规划基准上的实验表明，MGSD在4B和8B骨干网络上均持续提升视觉规划性能，宏平均分别提高了19.3%和18.4%。所得模型缩小了与符号输入上限的差距，而消融实验和诊断确认改进来源于视觉状态恢复和最优路径推理两方面。这些结果表明，模态差距感知自蒸馏不仅改善了模型感知可操作状态的方式，还提升了其在推断结构上进行规划的能力。代码见 https://github.com/Oranger-l/MGSD。

## Abstract
While vision-language models excel at general multimodal understanding, they still struggle with visual spatial planning. We attribute this to a perception-reasoning modality gap: visual planning requires models to infer latent state structures from pixels and then reason over the recovered structure to produce valid actions, whereas symbolic planning directly leverages explicit objects and constraints. This creates dual bottlenecks in visual state recovery and multi-step planning. To address this, we propose MGSD, a two-stage modality-gap-aware self-distillation framework. First, a cold-start grounding stage equips the visual student with reliable state representations, minimizing early perception noise. Second, a privileged teacher transfers planning capabilities via on-policy distillation, using explicit symbolic states to supervise the student's own visual rollout prefixes. Crucially, symbolic data is used strictly during training, leaving inference purely visual. Experiments on visual planning benchmarks show that MGSD consistently improves visual planning across both 4B and 8B backbones, raising the macro average by 19.3% and 18.4%, respectively. The resulting models narrow the gap to symbolic-input upper bounds, while ablations and diagnostics confirm that the improvement comes from both visual state recovery and optimal-path reasoning. These results suggest that modality-gap-aware self-distillation improves not only how models perceive actionable states, but also how they plan over the inferred structure. Code is available at https://github.com/Oranger-l/MGSD.

- **双重瓶颈**：视觉状态恢复（感知）和多步规划（推理）相互耦合，使得感知错误会严重破坏后续规划；即便感知完美，底层模型也可能存在规划能力不足。
- **现有方法不足**：标准SFT或RLVR难以分离感知与规划失败；现有视觉推理方法多在视觉模态内操作，未利用自然可得的符号状态作为特权监督。

## 2. 方法论：MGSD 框架
- **核心思想**：利用训练时配对的视觉-符号数据，通过两阶段自蒸馏将符号规划行为迁移到视觉学生模型，推理时完全依赖视觉输入。
- **两阶段设计**：
  - **阶段1：冷启动感知对齐（Perception‑Oriented SFT）**

  - **阶段2：符号引导的在线策略自蒸馏（OPSD）**  
    - 学生对视觉输入生成 rollout，冻结的文本教师以符号状态 T、参考规划 A 和学生前缀为条件，提供 token‑级分布监督；  
    - 优化反向KL式蒸馏损失：  
      Lsym = E_{y~πθ} [ Σ (log πθ(yt) – log πT(yt)) ]  （式2）  
    - 教师仅用于训练，推理时完全丢弃。

## 6. 主要结论与发现
- MGSD 在 4B 和 8B 骨干上分别将宏平均从 11.2→30.5（+19.3%）和 17.2→35.6（+18.4%），显著优于标准 SFT 和 GRPO。
- 相比于符号输入上限，MGSD‑4B 将差距缩小至 6.4 个百分点，表明大部分符号规划行为可被迁移到视觉推理。
- 诊断表明改进来源于**同时提升了视觉状态恢复和最优路径推理**，而非简单的感知或规划单方面提升。
- 冷启动感知对齐是 OPSD 生效的必要前提；冻结教师提供稳定蒸馏目标优于 EMA 教师；参考计划能为教师提供更精确的引导。

## 7. 方法优点
- **新颖的两阶段设计**：冷启动感知对齐 + 符号引导在线策略蒸馏，有效解耦感知与推理，并利用符号特权信息弥补模态差距。
- **推理无开销**：教师与符号上下文仅在训练时使用，推理时保持纯视觉输入，无需额外推理模块或符号生成。
- **诊断框架**：State F1 和 Plan on GT 的分解清晰度量了感知与推理各自的贡献，增强了分析可信度。
- **数据自动生成**：感知数据基于环境注释自动化构建，无需人工标注，具有可扩展性。

## 8. 不足与局限
- **数据依赖**：需要成对视觉-符号训练数据，在开放世界或真实场景中难以直接获取；依赖仿真环境的符号状态。
- **行动空间限制**：仅评估离散动作（L/D/R/U、PICK/DROP），未涉及连续控制、部分可观测、动态场景或长时域真实环境操作。
- **泛化风险**：在视觉歧义或分布偏移下，模型可能产生不忠实推理；未验证对噪声符号状态的鲁棒性。
- **实验范围**：仅在三个模拟环境上进行，任务复杂度有限（最大 8×8 网格），未能测试更复杂场景或真实图像。
- **算力开销**：OPSD 阶段需要在线 rollout 和教师逐 token 计算，训练成本高于普通 SFT。
