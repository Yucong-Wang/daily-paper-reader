## 摘要
语言引导的无人机智能体需在执行长时语义指令的同时生成平滑、物理可行的连续飞行指令，然而现有视觉-语言导航（VLN）基准通常使用离散或粗粒度动作，而现有无人机视觉-语言-动作（VLA）任务聚焦于短时、原子级机动。为填补无人机任务设置中的这一空白，我们引入了\textbf{FLIGHT}，一个面向混合无人机导航与推理任务的\textbf{精细化长时指令引导基准}，它结合了多阶段指令与密集的六自由度轨迹标注，包含两个数据集子集：精细化VLN与长时域流。为使无人机智能体在飞行中具备对任务执行状态和任务规划的实时推理能力，同时兼顾高频、实时精确控制，我们进一步提出了\textbf{FLIGHT VLA}，一种异步架构，将用于任务状态推理的低频流式飞行员视觉-语言模型（VLM）与用于连续控制的高频扩散动作模型解耦，并由明确描述当前飞行状态并预期下一子目标的\textbf{飞行员推理}文本进行监督。在闭环评估中，FLIGHT VLA在我们的FLIGHT基准上持续超越代表性VLN和VLA基线，实现了更强的多阶段完成、子目标遵循和终端控制能力。其训练的流式飞行员推理VLM进一步提升了无人机视频推理性能，验证了我们设计的有效性。

## Abstract
Language-guided UAV agents must execute long-horizon semantic instructions while producing smooth, physically feasible continuous flight commands, yet existing Vision-Language Navigation (VLN) benchmarks typically use discrete or coarse actions and existing UAV Vision-Language-Action (VLA) tasks focus on short, atomic maneuvers. To address this gap in UAV task settings, we introduce \textbf{FLIGHT}, a \textbf{F}ine-grained \textbf{L}ong-horizon \textbf{I}nstruction-\textbf{G}uided benchmark for \textbf{H}ybrid UAV navigation and reasoning \textbf{T}asks, which combines multi-stage instructions with dense 6-DoF trajectory annotations across two dataset splits: Fine-grained VLN and Long-horizon Flow. To endow the UAV agent with the capability of real-time in-flight reasoning over task execution status and mission planning, while simultaneously accommodating high-frequency, real-time precise control, we further propose \textbf{FLIGHT VLA}, an asynchronous architecture that decouples a low-frequency Streaming Pilot Vision-Language Model (VLM) for task-state reasoning from a high-frequency diffusion action model for continuous control, supervised by explicit \textbf{Pilot Reasoning} texts that summarize the current flight state and anticipate the next subgoal. In closed-loop evaluation, FLIGHT VLA consistently surpasses representative VLN and VLA baselines on our FLIGHT benchmarks, achieving stronger multi-stage completion, subgoal adherence, and terminal control. Its trained Streaming Pilot Reasoning VLM further improves UAV video reasoning, validating the effectiveness of our design.

## 论文结构化总结：像飞行员一样思考：精细化长时域无人机导航

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题：** 现有的语言引导无人机（UAV）任务设定存在两极分化：
    - **视觉-语言导航 (VLN)** 任务侧重于长距离、粗粒度的“目的地寻找”，通常使用离散动作或简化模型，无法满足真实无人机飞行对连续、平滑轨迹和三维运动学约束的需求。
    - **视觉-语言-动作 (VLA)** 任务虽然实现了连续控制，但仅限于短时、原子级的单一动作（如绕飞、降落），缺乏多阶段、长时域的任务规划与执行能力。
- **研究动机：** 真实的无人机复杂任务（例如“飞过两个街区，在路口左转，在空中悬停投递包裹，最后降落在第二个邮箱旁”）**无法**被简化为单一的导航或短时控制问题。这类任务要求无人机必须同时具备：
    1.  **理解长时、多阶段语义指令**并持续追踪任务状态（如当前子目标、已完成阶段、下一步意图）。
    2.  **生成精确、平滑、物理可行的连续控制命令**。
- **核心挑战：**
    1.  **长时域下的状态保持困难：** 在长时间飞行中，由于视点变化剧烈，模型容易出现语义漂移、子目标遗忘或任务完成状态误判。
    2.  **语义理解与实时控制的时序鸿沟：** 复杂语义推理（VLM）计算量大、频率低，无法满足高频、低延迟的连续控制需求。
- **整体含义：** 语言引导的无人机自主导航必须从根本上调和“长时语义规划”与“高频连续控制”之间的时序不匹配问题。本文旨在填补这一研究空白，提出新的基准与模型架构。

### 2. 论文提出的方法论：核心思想、关键技术细节

本文提出了一套包含**基准**（FLIGHT）与**模型**（FLIGHT VLA）的完整方案，其核心思想是**通过“任务分解”与“异步解耦”来弥合高层规划与低层控制之间的鸿沟**。

#### 2.1. FLIGHT 基准与数据集

- **任务设定：**
    - **精细化VLN (Fine-grained VLN)：** 在传统VLN基础上，增加了更密集的指令粒度（如“经过花坛”、“沿喷泉右侧移动”）和连续动作空间（6-DoF轨迹点），要求模型在长程导航中也能进行精细控制。
    - **长时域流 (Long-horizon Flow)：** 继承了UAV-Flow的原子动作概念，但扩展为多阶段组合任务（如“先绕树，再穿越，最后降落”）。
    - **飞行员推理任务 (Pilot Reasoning Task)：** 作为一个独立的VLM视频推理基准，要求模型根据视频和指令，推理出当前状态和下一动作。
- **数据收集与标注（关键技术细节）：**
    1.  在Unrealzoo仿真环境中，由人类专家飞行员录制6-DoF连续轨迹和第一视角视频。
    2.  **VLM辅助的分段语义标注：** 将视频按时间分段，并用VLM（以无人机姿态数据作为辅助提示）生成每个片段的精细语义标签（如“在十字路口左转”）。
    3.  **指令生成：** 利用LLM将多个分段语义标签合并为连贯、自然的多阶段指令，避免模板化。
    4.  **飞行员推理文本生成：** 利用“未来信息”（下一段的语义标签）作为先验，指导VLM生成描述当前状态并预期下一步行动的推理文本，这相当于显式化了人类的决策链。

#### 2.2. FLIGHT VLA 异步架构

核心思想是**异步解耦**，将策略分解为两个不同时间尺度上运作的模块：
`π : (S_t, V_[t-W,t], I, M_t) -> a_{t+1}, M_{t+1}`

- **低频流式飞行员VLM (Streaming Pilot VLM) - “慢系统”：**
    - **功能：** 负责任务在线推理、状态识别和高级规划。以约0.5~1Hz的频率运行。
    - **工作原理：** 每轮推理接收任务指令`I`、最新的视频/状态窗口和历史推理记忆`M`，生成新的飞行员推理文本`M_τ`和隐层语义特征`z_τ`。该特征编码了指令、上下文、当前状态和未来计划，用于指导动作生成。
    - **训练：** 使用**重叠分块全注意力**策略，在连续视频流上模拟推理时的流式处理，并用FLIGHT数据集中的“飞行员推理”文本进行监督。
- **高频扩散动作模型 (Diffusion Action Model) - “快系统”：**
    - **功能：** 负责高频、连续的控制动作生成。以约10~30Hz的频率运行。
    - **工作原理：** 每个控制步接收：
        1.  **实时视觉反馈**：由独立视觉编码器从当前帧`V_t`提取特征`f_t`。
        2.  **相对位姿编码**：将当前无人机位姿`q_t`表示为相对于VLM推理时刻的坐标系。
        3.  **最新语义特征**：从VLM异步获取的`z_τ(t)`。
        4.  将以上信息拼接作为条件`c_t`，输入DiT（Diffusion Transformer）直接生成未来H步的连续动作序列`A_t`。
- **联合训练与时延对齐 (关键技术细节)：**
    - 训练损失函数：`L = λ_vlm * L_vlm + λ_act * L_act`。
    - **时延感知训练策略：** 在训练动作模型时，不是为了它提供“当前时刻”的VLM特征，而是模拟推理延迟，提供**不同历史时间偏移**（如1秒前、2秒前）的VLM特征。这使动作模型学会在接收到“过时”语义特征时，利用实时视觉和状态信息进行补偿，增强了系统的鲁棒性。

### 3. 实验设计：数据集、基准与对比方法

- **数据集：**
    - **FLIGHT-Long-horizon Flow：** 包含4,098条轨迹，每条轨迹平均3.37个动作实例，平均距离32.8米。
    - **FLIGHT-Fine-grained VLN：** 包含6,689条轨迹，每10Hz采样一次动作，平均每轨迹475个动作，平均距离154.5米。
- **闭环评估基准 (Benchmark)：**
    - **长时域流任务指标：** 成功率（SR）、阶段成功率（SSR）、归一化动态时间规整（NDTW）。
    - **精细化VLN任务指标：** 导航误差（NE）、成功率（SR）、Oracle成功率（OSR）、NDTW，以及新提出的**指令遵循成功率（IASR）**，用于严格评估子任务的顺序执行情况。
    - **飞行员推理VLM准确性：** 使用GPT-5评估预测的推理文本与真实文本的语义一致性。
- **对比方法 (Baselines)：**
    - **VLA模型：**
        - OpenVLA-UAV：离散化动作空间的VLA模型。
        - Memory VLA：引入记忆结构用于长时任务的VLA模型。
    - **VLN模型：**
        - CMA+LAG：跨模态注意力模型，用于离散导航。
        - NaVid：基于视频的VLN大模型。

### 6. 论文的主要结论与发现

- **现有方法在长时域、细粒度UAV任务上表现不佳：** VLN方法（如CMA+LAG、NaVid）由于离散动作和上下文限制，无法完成精细、连续的控制及多阶段任务。VLA方法（如OpenVLA、Memory VLA）在处理超过两个步骤的复合任务时，性能显著下降。
- **FLIGHT VLA 方法有效：** 在**所有指标**上，FLIGHT VLA均显著优于所有基线方法。
    - **长时域流：** FLIGHT VLA的SR（59.0%）远超Memory VLA（18.5%），SSR（78.97%）也大幅领先（30.87%/38.69%）。
    - **精细化VLN：** FLIGHT VLA在IASR（11.0%）上同样远超NaVid（5.0%）和CMA+LAG（4.5%），证明了其在精细指令遵循上的优势。
- **Pilot Reasoning 训练提升VLM能力：** 经过流式推理范式训练的3B模型在**状态预测**（64.2%）和**动作预测**（68.1%）准确性上，远超大参数的商业模型（如Gemini-3-flash，分别为25.8%/39.1%）。
- **结论：** 显式的中间推理（Pilot Reasoning）和异步快慢系统（FLIGHT VLA）设计，是解决语言引导的无人机长时域、精细化控制问题的有效途径。

### 7. 优点：方法或实验设计上的亮点

- **创新性的问题提出与解决方案：** 精准识别了“VLN”与“VLA”任务之间的空白，并提出了一个兼顾两者优点的“混合任务”设定。
- **精巧的异步架构设计：** FLIGHT VLA的“快-慢”异步解耦思想，直接针对核心挑战（语义推理与实时控制的时序鸿沟），是该工作的最大亮点。它并非简单地将VLM输出作为动作输入，而是通过隐层特征传递和时延感知训练，实现了两者的高效协同。
- **显式化推理过程：** “飞行员推理”（Pilot Reasoning）不仅是训练数据，更是认知锚点，将人类的决策链外部化，为模型提供了更丰富的语义监督，显著增强了模型的可解释性和任务追踪能力。
- **严格的评估指标：** 提出的**指令遵循成功率（IASR）** 是对传统VLN仅评估最终位置的成功/失败判据的有力补充，能更精确地衡量模型对多阶段指令的忠实执行程度，更贴合实际应用场景。
- **数据标注流程：** 结合VLM辅助和人类验证的自动化标注流程，确保了大规模数据集的质量和语义丰富度，特别是利用运动数据作为VLM提示，弥补了纯视觉模型对空间运动理解不足的缺陷。

### 8. 不足与局限：包括实验覆盖、偏差风险、应用限制等

- **仿真到真实的鸿沟（Sim-to-Real Gap）：** 所有实验均在Unrealzoo仿真环境中进行。模型完全依赖于仿真环境下的视觉渲染和物理引擎，未在真实世界的无人机硬件上进行验证。仿真环境的传感器噪声、光照、物体纹理等与现实有差异，可能导致模型在真实场景中性能下降。
- **计算开销：** 尽管采用了异步架构，但FLIGHT VLA仍依赖一个3B的VLM进行频繁的流式推理，即使频率较低，整体计算成本和对边缘部署设备的算力要求仍然较高，可能限制其在实际无人机平台上的应用。
- **对“飞行员推理”标注的依赖：** 模型的训练高度依赖于人工/ VLM生成的“飞行员推理”文本。这种显式标签的获取成本高，且在复杂、未见过场景下，如何自动生成高质量推理文本仍有挑战。
- **动作空间的简化：** 尽管是连续动作，但论文简化为4维（∆x, ∆y, ∆z, ∆ω）的相对位姿命令。这忽略了无人机完整的动力学特性（如加速度、角速度、推力等），生成的轨迹对于真实飞机的底层飞控系统可能并非最优或最可执行的。这是一个“任务级规划”向“动作级命令”的映射，而非真正的“底层控制”。
- **评估指标的局限：** 在精细化VLN任务上，SR的最高成绩仅为13.5%（FLIGHT VLA），整体导航成功率很低，表明长程导航任务仍极具挑战。模型虽然在IASR上显著优于基线，但绝对值（11%）仍然不高，说明对复杂指令的精细理解和执行能力仍有巨大提升空间。
