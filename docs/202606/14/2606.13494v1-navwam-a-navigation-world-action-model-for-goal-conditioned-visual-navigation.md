## 摘要
目标条件视觉导航要求机器人在部分可观测条件下通过预测其运动如何改变未来的自我中心视图以及这种变化是否使其更接近目标来采取行动。导航世界模型提供了这种视觉预见能力，但它们仍然是预测模块，需要外部规划器将预测的未来转换为闭环控制。我们提出了导航世界动作模型（NavWAM），这是一种扩散变换器策略，通过将未来观测、目标进展值和动作块表示在共享的潜在序列中，将导航世界模型预测转化为可执行的动作。通过联合学习未来预测以及决定闭环行为的动作和价值目标，NavWAM使视觉预见可直接用于机器人控制。我们通过模拟预训练和真实机器人自适应构建NavWAM，并在图像目标导航任务上将其与基于规划的世界模型和代表性的直接导航策略进行评估。在离线基准测试和闭环真实机器人部署中，NavWAM在使用默认策略模式且无需CEM风格动作搜索的情况下，在我们的评估中优于基于规划的世界模型基线。项目页面：https://dachii-azm.github.io/navwam/

## Abstract
Goal-conditioned visual navigation requires a robot to act under partial observability by anticipating how its motion will change the future egocentric view and whether that change brings it closer to the goal. Navigation world models provide such visual foresight, but they remain prediction modules that require an external planner to convert predicted futures into closed-loop control. We propose Navigation World Action Model (NavWAM), a diffusion-transformer policy that turns navigation world-model prediction into executable action by representing future observations, goal-progress values, and action chunks in a shared latent sequence. By learning future prediction jointly with the action and value targets that determine closed-loop behavior, NavWAM makes visual foresight directly usable for robot control. We build NavWAM through simulation pretraining and real-robot adaptation, and evaluate it on image-goal navigation against planning-based world models and a representative direct navigation policy. Across offline benchmarks and closed-loop real-robot deployment, NavWAM improves over planning-based world-model baselines in our evaluations while using the default policy mode without CEM-style action search. ## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：目标条件视觉导航（Goal-conditioned Visual Navigation）需要在部分可观测的环境下，机器人不仅要根据当前观测采取行动，还要预测运动后未来的自我中心视图（egocentric view）是否更接近目标。现有方法分为两类：直接预测动作的策略（如扩散策略、VLA模型）虽然高效，但未显式学习预测未来视觉后果；而导航世界模型（Navigation World Models, NWMs）可以预测未来视图，但仅作为预测模块，需要外部规划器（如交叉熵方法CEM）进行测试时的轨迹优化才能转化为闭环控制。这种分离导致推理计算量大、闭环行为依赖外部搜索。
- **研究动机**：如何将显式的视觉预见（visual foresight）直接用于可执行的闭环控制，避免外部规划器带来的瓶颈？
- **整体含义**：本文提出NavWAM (Navigation World Action Model)，将未来观测、目标进展值（goal-progress value）和动作块（action chunk）编码为共享的潜在序列，通过扩散变换器联合学习，使视觉预见直接驱动闭环导航策略。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将导航世界模型的预测任务和动作生成任务统一为一个联合去噪问题。NavWAM使用一个固定的9帧潜在画布（latent canvas），其中：
  - **条件帧（观测）**：空白帧、当前机器人状态、目标图像、当前观测图像。
  - **预测帧（生成）**：动作块（H步动作序列）、未来状态、两个未来观测图像、标量目标进展值。
- **关键技术细节**：
  - **骨干网络**：基于Cosmos Predict2 (2B参数) 扩散变换器（diffusion transformer），利用其因果VAE和DiT骨干。
  - **非图像变量处理**：状态、动作块、标量值被归一化后广播到潜在空间网格，通过空间平均解码。
  - **多模式训练**：训练时每个样本随机分配三种模式之一：策略模式（50%，条件+预测全部帧）、世界模型模式（25%，额外条件动作）、价值模式（25%，条件所有未来帧仅预测价值）。单一权重同时学习动作生成、未来预测和价值估计。
  - **损失函数**：扩散去噪损失，并对动作帧施加更高权重（λ=5）以避免高维图像重建损失主导。
  - **推理**：默认只使用策略模式，以单次去噪链直接输出动作块，以滚动时域方式执行。未来视图和价值作为可解释的附带输出。可选的价值引导最佳采样（best-of-N）未在主结果中使用。
- **公式**：联合世界动作预测形式化为 \( p_\theta(a_{t:t+H-1}, s_{t+H}, o_{t+H-1:t+H}, v_{t+H} | o_t, g) \)，其中 \(v_{t+H}\) 是基于轨迹末端距离计算的归一化目标进展值。

## 6. 论文的主要结论与发现

- NavWAM在默认策略模式下（无CEM搜索）即可显著优于NWM等基于规划的世界模型基线（GO STANFORD上ATE从0.453降至0.324，微调后0.192）。
- 将视觉预测与动作、价值联合学习是关键：单独图像预测不足以导航，加入动作/状态/价值后性能大幅提升。
- 联合预测保留了未来预测能力：NavWAM的主体一致性（0.668）高于NWM（0.524）。
- 与更大直接策略OmniVLA（7B参数）相比，NavWAM（2B参数）在SIT上ATE相当甚至更好，且额外提供未来视图和价值预测。
- 真实机器人部署中，NavWAM成功率79.2%（19/24），高于OmniVLA的58.3%和NWM的16.7%。
- 结论：视觉预见在导航中最有用时与动作和价值目标联合学习，而非作为独立规划模块。

## 7. 优点
