## 摘要
动态环境中的终生具身导航要求机器人从碎片化观测中形成持久的场景理解，这对于依赖显式地图或场景图且在结构化环境之外难以泛化的现有方法仍然困难。我们提出AllDayNav，一种终生自我学习导航框架，通过强化学习将场景动态隐式编码到大模型的十亿级参数中，并由自演化多模态记忆驱动，该记忆维护并更新视觉关键帧、语义描述和时间上下文，同时自主生成开放词汇指令、图像目标和结构化奖励。在跨房间、跨回合和跨任务场景的合成及真实世界环境中的实验表明，AllDayNav的成功率接近100%，并在路径效率和鲁棒性上持续超越基于地图、VLM和RL的强基线，证明了隐式、记忆驱动的强化学习作为可靠终生导航的可扩展替代方案，以替代显式映射。

## Abstract
Lifelong embodied navigation in dynamic environments requires robots to form persistent scene understanding from fragmentary observations, which remains difficult for existing methods that rely on explicit maps or scene graphs and struggle to generalize beyond structured settings. We propose AllDayNav, a lifelong self-learning navigation framework that implicitly encodes scene dynamics into the billion-scale parameters of a large model via reinforcement learning, powered by a self-evolving multimodal memory that maintains and updates visual keyframes, semantic descriptions, and temporal context while autonomously generating open-vocabulary instructions, image goals, and structured rewards. Experiments in both synthetic and real-world environments across cross-room, cross-episode, and cross-task scenarios show that AllDayNav achieves success rates approaching $100\%$ and consistently surpasses strong map-based, VLM, and RL baselines in path efficiency and robustness, demonstrating implicit, memory-driven reinforcement learning as a scalable alternative to explicit mapping for reliable lifelong navigation.

## 2. 论文提出的方法论
### 核心思想
- **隐式场景编码**：通过强化学习（RL）将场景动态隐式编码到大模型（VLA）的十亿级参数中，替代显式地图。
- **记忆-策略协同进化**：构建自演化多模态记忆数据库，维护视觉关键帧、语义描述和时间上下文；记忆自动生成自指令（self-instruction）并检索视觉目标，驱动策略无监督训练。

### 关键技术细节
1. **VLA导航骨干**：基于Qwen2-7B Transformer，采用双编码器（SigLIP + DINOv2）融合语义和结构特征；历史帧通过全局平均池化压缩，并施加指数衰减注意力偏置；动作预测为8个连续航点（waypoint）。
2. **自演化多模态记忆数据库**：每个条目包含RGB关键帧、VLM生成的语义描述、时间戳和LLaVE嵌入。探索中检测视觉变化（余弦相似度阈值0.15）筛选候选帧，与现有记忆去重（阈值0.7），若相似则通过图像-文本对齐度决定是否替换。
3. **自指令生成与混合目标检索**：每10回合或记忆增长20%时，调用Qwen3-VL-8B-Instruct基于记忆描述生成50条开放词汇指令（覆盖目标、功能、空间、组成、时间等类型）。检索时优先从记忆提取，置信度低于0.6或早期随机回退时使用互联网图片库（Ginternet）作为目标。
4. **在线强化学习（保守Q学习，CQL）**：奖励函数包括成功奖励（+10，视觉相似度≥0.75）、渐进奖励（首次超过0.5/0.6/0.7时+2）、时间惩罚（-0.01/步）、碰撞惩罚（-0.1）。CQL目标函数包含Bellman误差和保守正则化项（α=5），策略优化结合Q值最大化和行为克隆正则化（λ_bc=0.1）。

### 公式/算法流程（文字说明）
- 预训练阶段：使用50,000条专家轨迹（80%最短路径 + 20%随机探索）进行监督学习（MSE损失）。
- 在线阶段：算法循环进行主动探索（A过程）和策略优化（B过程）。探索中构建记忆、生成自指令、执行当前策略并收集过渡数据；优化阶段从回放缓冲区采样，交替更新评论家网络（CQL损失）和策略网络。
