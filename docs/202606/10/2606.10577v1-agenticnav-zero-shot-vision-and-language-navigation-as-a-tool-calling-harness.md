## 摘要
连续环境下的零样本视觉与语言导航（VLN-CE）最近随着大型视觉语言模型（VLM）的发展而变得可行。然而，现有方法通常依赖学习到的航路点预测器来提出可执行的动作，这严重限制了模型的行动空间，并且未能有效利用深度输入。此外，记忆通常通过积累包含大量无关上下文的冗长文本或视觉历史，或者通过检索跨集经历来处理，这削弱了零样本设置。在本文中，我们将零样本VLN-CE重新构想为VLM与环境之间的智能接口，并提出了AgenticNav，这是一个轻量级的导航框架，将动作、深度和记忆作为可调用的工具暴露出来。动作工具允许VLM直接从RGB观测中选择目标像素，并将其转换为可执行的移动，而不是从预测的航路点中选择。深度通过按需像素深度工具暴露，使得VLM能够在关键位置请求精确的公制距离。对于记忆，AgenticNav提供了一个紧凑的地图图像来总结历史轨迹，并配有一个回忆工具，允许VLM有选择地重新访问过去的视觉观测，而不会使提示上下文过载。在R2R-CE基准测试中，在相同的VLM骨干网络下，AgenticNav在零样本方法中建立了新的最先进（SOTA）性能。实际验证进一步突出了其与先前方法相比的零样本泛化能力。消融实验表明，我们的动作工具设计优于传统的航路点预测器，并且深度工具和智能记忆进一步提升了导航性能。

## Abstract
Zero-shot vision-and-language navigation in continuous environments (VLN-CE) has recently become feasible with large vision-language models (VLMs). However, existing methods typically rely on learned waypoint predictors to propose navigable actions. This severely limits the model's action space and fails to leverage depth inputs effectively. Moreover, memory is commonly handled by accumulating long textual or visual histories with substantial irrelevant context, or by retrieving cross-episode experiences, which weakens the zero-shot setting. In this paper, we rethink zero-shot VLN-CE as an agentic interface between the VLM and the environment, and present AgenticNav, a lightweight navigation harness that exposes action, depth, and memory as callable tools. Instead of choosing from predicted waypoints, the action tool allows the VLM to directly select a target pixel in RGB observations, converting it into executable motion. Depth is exposed through an on-demand pixel-depth tool, enabling the VLM to request precise metric distances only where they matter. For memory, AgenticNav provides a compact map image summarizing the historical trajectory, paired with a recall tool that allows the VLM to selectively revisit past visual observations without overwhelming the prompt context. On the R2R-CE benchmark, AgenticNav establishes new state-of-the-art (SOTA) performance among zero-shot methods given the same VLM backbone. Real-world validation further highlights its zero-shot generalization compared to prior methods. Ablations show that our action tool design outperforms traditional waypoint predictors, and that depth tool and agentic memory further contribute to navigation performance.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究问题**：连续环境下的零样本视觉与语言导航（VLN-CE）要求智能体在未见过的3D场景中，仅根据自然语言指令导航，无需预定义导航图或任务特定训练。现有零样本方法通常依赖学习到的航路点预测器来生成候选动作，这限制了VLM的动作空间（只能选择预测的航点，无法自由选择任意可见位置），且未能有效利用深度信息（深度被隐藏在预测器中，VLM无法直接获取精确度量）。此外，内存管理常用累积冗长的文本/视觉历史（上下文过长、噪声多）或检索跨集经验（削弱零样本假设），导致性能瓶颈。
- **整体含义**：本文认为零样本VLN-CE的限制不仅在于VLM本身，更在于连接VLM与环境的接口设计。通过将导航重构为工具调用过程，可以突破传统接口的约束，实现更灵活、高效、零样本的导航。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
将零样本VLN-CE建模为VLM与环境之间的智能工具调用框架（Agentic Harness），暴露三个关键调用工具：**动作工具**、**深度工具**、**记忆工具**。VLM通过调用这些工具进行决策，而工具负责将VLM的选择转化为可执行指令。

### 关键技术细节

- **动作工具 (Action Tool: `move_to(k, u, v)`)**  
  - VLM直接在RGB图像中选择一个像素点（指定视图索引k和归一化坐标u,v），而不是从有限航点中选择。
  - 工具对该像素进行深度反投影（depth unprojection）计算3D点，并通过安全检查（检查机器人周围矩形走廊内是否有障碍物）决定是否执行。
  - 若安全，则输出可执行的相对偏航角θ和前进距离ρ；否则返回`Reselect`要求重选。

- **深度工具 (Depth Tool: `query_depth(P)`)**  
  - VLM可以按需批量查询候选像素的度量深度，返回：
    - 像素处的度量深度 `d_i`
    - 局部5×5邻域深度统计（最小值、均值、中位数）帮助判断是否为稳定表面
    - 动作预览（若该像素被选中将产生的θ和ρ）
  - 不提供完整深度图，只提供VLM指定的关键位置深度，避免VLM解析密集深度图。

- **智能记忆与回忆工具 (Agentic Memory & Recall Tool: `recall(p, k)`)**  
  - 维护一个紧凑的轨迹地图图像 `B_t`（在提示中默认包含）和一个有限的历史观测缓存 `C_t`（不默认插入提示）。
  - VLM调用 `recall` 工具可以回溯过去某个决策点的特定视图图像，仅将相关视觉信息加载到上下文，避免累积无关历史帧。

- **停止工具 (stop)**：终止导航。

**整体流程**：在每个决策步，VLM接收多视图RGB、地图图像、当前对话/工具历史，可以多次调用深度工具和回忆工具获取证据，最终调用动作工具或停止工具。动作执行后进入下一个观测。

## 6. 主要结论与发现

- AgenticNav在R2R-CE上取得零样本SOTA：AgenticNav-GPT-5.5达到55% SR（相比SmartWay-GPT-5.5的44%提升11个百分点）和48.41% SPL（提升13.37个百分点）。Gemini-2.5-Pro骨干下也优于Open-Nav和EvoNav（在纯零样本设定下）。
