## Abstract
In this report, we present Hy-Embodied-0.5-VLA, abbreviated as HyVLA-0.5, an end-to-end system that spans the full robot learning stack: data collection, model design, continued pre-training and supervised fine-tuning, RL post-training, and real-world deployment. Each component serves a distinct role in this stack.

### 论文核心问题与整体含义（研究动机和背景）

- **核心问题**：现有视觉-语言-动作（VLA）模型主要聚焦于策略本身的改进，而忽略了从数据收集到真实部署的完整系统整合。这导致三个耦合挑战：  
  1. **数据采集**：传统遥操作缺乏力反馈，无法处理精细操作；人类演示虽有多样性但动作标签粗糙；现有UMI设备依赖SLAM，精度不足且设备笨重。  
  2. **模型设计**：通用VLM骨干未针对机器人控制设计；离散化动作空间限制控制频率与精度；标准模仿学习难以达到最后一步的灵巧性。  
  3. **部署执行**：高容量VLA模型推理速度慢于伺服周期，且动作块之间存在不连续性。  

- **整体含义**：真正的通用机器人需要端到端的学习堆栈，而非孤立的模型。HyVLA-0.5 旨在通过协同设计数据、建模、策略细化与部署，构建一个可部署的通用机器人系统。

### 论文提出的方法论：核心思想、关键技术细节

#### 核心思想
- 构建一个覆盖 **数据收集 → 模型设计 → 持续预训练与SFT → RL后训练 → 真实部署** 的完整流水线，将VLA从单一模型转变为实用的机器人学习堆栈。

#### 关键技术细节

1. **数据收集 (Hy-UMI-10K)**  
   - 定制指尖UMI设备：采用光学动捕系统（替代SLAM）实现亚毫米级6-DoF轨迹跟踪；手指绑式执行器恢复自然力反馈；头戴式第一人称视角摄像头提供全局语义。  
   - 收集超过10,000小时、100万条演示，覆盖70个任务（洗衣房、厨房、个人护理等6大场景族）。  
   - 数据可直接用于后训练，减少目标机器人数据采集需求。

2. **模型架构**  
   - **骨干网络**：Hy-Embodied-0.5-MoT（4B参数，Mixture-of-Transformers），原生分辨率图像编码（Hy-ViT 2.0），支持单帧和多帧（K=6）输入。  
   - **动作专家**：采用条件流匹配（Flow Matching）生成连续动作块（H=50）。双塔路由：VLM塔处理视觉/语言，动作专家塔处理状态与噪声动作，通过共享自注意力交互。  
   - **紧凑记忆编码器**：在ViT中每隔L层插入因果时序注意力，将历史帧信息压缩到当前帧表示，不增加新参数（复用QKV投影）。  
   - **增量动作块表示 (delta-chunk)**：预测相对末端执行器帧的增量动作（10维/臂：3D平移+6D连续旋转+1D夹爪），解耦策略学习与具体运动学。

3. **训练流程**  
   - **持续预训练**：在10K小时UMI数据上训练200K步（batch size 1024，学习率5e-5），仅流匹配损失，K=1（单帧）。  
   - **监督微调 (SFT)**：两个轨道：  
     - Track A：目标机器人演示微调并部署在同体（Dobot X-Trainer，4任务，300演示/任务）。  
     - Track B：仅用UMI演示微调，跨体迁移（JAKA K1，Astribot S1，无目标机器人遥操作）。设置K=6，历史间隔1秒。

4. **RL后训练（FlowPRO）**  
   - **无批评家、无奖励模型**：通过遥操作干预-回滚管道收集正负轨迹对，使用平滑插值生成密集逐状态偏好元组。  
   - **RPRO损失**：基于流匹配的隐式奖励 + 对称近端正则化，防止奖励黑客；对比梯度抵消特性允许安全合并SFT样本。  
   - **迭代优化**：3轮迭代，每轮25K步，混合新偏好数据、历史数据和SFT数据（比例70/15/15）。

5. **部署**  
   - **跨体平台映射**：增量动作块通过正向运动学或启发式（人形机器人）转为世界坐标，保持不变的动作接口。  
   - **异步推理**：生产者-消费者运行时，推理与执行重叠。  
   - **贝塞尔块拼接**：丢弃陈旧前缀，用三次贝塞尔曲线保证C1连续性（位置R³，姿态SLERP，夹爪线性插值）。

### 论文的主要结论与发现

1. **完整堆栈有效**：HyVLA-0.5在模拟（RoboTwin 2.0 Clean 90.9%/Randomized 90.1%）和真实世界（Track A/B显著优于π0/π0.5）均达到最先进水平。
2. **UMI预训练关键**：大型高精度UMI预训练显著提升精密操作瓶颈（如折叠、拉链）成功率，且跨体迁移中无目标机器人遥操作时仍大幅领先。
3. **FlowPRO高效**：无需奖励模型，通过偏好优化将失败转化为提升，在三轮迭代后任务成功率接近天花板（>98%），且完成时间更短。
4. **紧凑记忆编码器**：提供一致但有限的性能增益（模拟+2.1/+1.5点）。
5. **增量动作块+平台映射**：成功实现从UMI数据到异构机器人（固定臂、人形机器人）的零调试迁移。

### 优点：方法或实验设计上的亮点
