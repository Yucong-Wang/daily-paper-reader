## 摘要
目标目标导航（Object-Goal Navigation, ObjNav）要求具身智能体仅依靠自我中心的视觉观测自主定位指定目标。现有单一方法难以进行长时程推理，且对陌生环境的泛化能力较差。为解决这些局限，我们提出SAGE-Nav，一种新颖的分层框架，将大语言模型（LLM）的推理能力与动态场景图相结合。关键之处在于，它将异步的全局语义规划与高频反应式控制回路解耦。LLM作为全局规划器，将抽象指令分解为一系列语义锚定的航点。为了将这些规划转化为密集的多模态引导，我们设计了分层场景图编码器（HSGE），利用关系图卷积生成保留语义和空间拓扑的结构感知嵌入。此外，我们开发了目标感知对齐融合网络（GAFN），将实时感知与这些结构先验动态融合。通过使用具有显式归纳偏好的自适应门控机制，GAFN确保低级策略具备稳健的视觉-拓扑对齐。在i-THOR和RoboTHOR环境中的大量评估表明，SAGE-Nav达到了最先进的性能，在导航效率和零样本泛化方面取得了显著提升，同时保持了物理机器人部署所需的低控制延迟。

## Abstract
Object-Goal Navigation (ObjNav) requires embodied agents to autonomously locate specified targets using only egocentric visual observations. Existing monolithic methods struggle with long-horizon reasoning and generalize poorly to novel environments. To address these limitations, we propose SAGE-Nav, a novel hierarchical framework that integrates the reasoning capabilities of Large Language Models (LLMs) with dynamic scene graphs. Crucially, it decouples asynchronous global semantic planning from the high-frequency reactive control loop. The LLM serves as a global planner, decomposing abstract instructions into a sequence of semantically grounded waypoints. To translate these plans into dense multi-modal guidance, we design a Hierarchical Scene Graph Encoder (HSGE) that leverages relational graph convolutions to produce structure-aware embeddings preserving both semantic and spatial topology. Furthermore, we develop the Goal-aware Alignment-Fusion Network (GAFN) to dynamically fuse real-time perception with these structural priors. Using an adaptive gating mechanism with an explicit inductive bias, GAFN ensures robust visual-topological alignment for the low-level policy. Extensive evaluations in the i-THOR and RoboTHOR environments demonstrate that SAGE-Nav achieves state-of-the-art performance, delivering substantial gains in navigation efficiency and zero-shot generalization while maintaining the low control latency required for physical robotic deployment.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
构建分层的导航框架（SAGE-Nav），包含三个关键模块：
1. **LLM驱动的层级全局规划器（H-GP）**：基于层级场景图和检索增强生成（RAG），将抽象指令分解为语义锚定的航点序列。
2. **层级场景图编码器（HSGE）**：使用关系图卷积网络（R-GCN）将航点编码为结构感知的嵌入向量，保留语义和空间拓扑。
3. **目标感知对齐融合网络（GAFN）**：通过自适应门控机制动态融合实时视觉特征与结构先验，实现视觉-拓扑对齐。

### 关键技术细节
- **层级场景图构建**：节点分为物体级和聚簇级，边包含同层空间边和跨层层级边。场景图增量式构建，基于空间相似度（高斯核）和语义相似度（余弦相似度）进行节点合并。
- **检索增强子图提取**：计算每个节点与目标指令的语义相似度，并结合邻域信息进行空间增强，取Top-K节点构成紧凑子图。
- **LLM全局规划**：将航点序列问题建模为旅行商路径问题（TSPP），提出层级连贯权重 \(W(v_i, v_j) = \lambda_{\text{sem}} S_{\text{multi-level}}(v_i, v_j) - \lambda_{\text{dist}} D_{\text{geo}}(v_i, v_j)\)，用贪心TSPP启发式生成最优序列。同时支持对未见目标的假设推理（基于LLM的语义兼容性先验）。
- **HSGE编码**：对每个节点，输入为CLIP视觉嵌入与正弦编码3D坐标的拼接。使用3层R-GCN进行层内关系编码，残差连接保持语义。通过航点条件注意力进行对齐，最后聚合各层级节点得到航点嵌入 \(E_w\)。
- **GAFN融合**：先通过交叉注意力将结构嵌入注入视觉特征得到 \(F_a\)，然后计算实时语义对齐分数 \(\alpha_t = \cos(F_v, E_w)\)，通过门控向量 \(\lambda(\alpha_t)\) 自适应融合：\(F_f = \lambda \odot F_v + (1-\lambda) \odot F_a\)。门控设计引入显式归纳偏好：当对齐分数高时（接近目标），网络更依赖结构上下文。
- **强化学习优化**：使用A3C算法，奖励函数包括可见奖励、前进奖励、距离减小奖励和步数惩罚。LSTM Actor-Critic网络输出动作。

## 6. 论文的主要结论与发现

- SAGE-Nav在iTHOR上达到82.47% SR（ALL）和77.22% SR（L≥5），在RoboTHOR上达到52.35% SR（ALL）和40.82% SR（L≥5），均超过当时最优方法（TSOG、CGI-GAIL等）。

- 性能-效率分析表明，SAGE-Nav在保持竞争性SR的同时，拥有最低的每步延迟（0.42秒）和最少的LLM调用次数（平均9.1次/集），实现了最优的SPL（30.1%）。

- 失败案例主要为：目标不可见时终止、语义相似物体检测错误、陷入复杂布局、过早终止。揭示了当前模型在主动视角调整和深度障碍物感知方面的不足。
