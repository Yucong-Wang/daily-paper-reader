## 摘要
巡视器依赖感知来维护空间地图，该地图编码了物体和传感器质量（例如，距离可靠性、光照伪影、数据密度），从而在部分可观测条件下引导数据融合、嵌入更新和导航。为了研究这些耦合的感知-导航过程，我们提出了CrossMaps，一种实时置信度感知的开放词汇语义建图流水线，它从RGB-D数据构建可语言查询的地图。基于VLMaps风格的方法，CrossMaps将多尺度CLIP嵌入与置信度感知融合以及由短期记忆（STM）和长期记忆（LTM）组成的双记忆架构相结合。STM利用几何、语义和时间置信度线索聚合噪声视觉观测，而置信度高且一致的单元则被提升到LTM作为持久语义地标。CrossMaps专为在Jetson Orin驱动的UGV上部署而设计，与SLAM协同工作，实时运行并生成语义热图，可通过自然语言查询来指导巡视器导航。

## Abstract
Rovers rely on perception to maintain spatial maps that encode both objects and sensor quality (e.g., range reliability, lighting artifacts, data density), guiding data fusion, embedding updates, and navigation under partial observability. To study these coupled perception-navigation processes, we present CrossMaps, a real-time confidence-aware open-vocabulary semantic mapping pipeline that constructs language-queryable maps from RGB-D data. Building on VLMaps-style approaches, CrossMaps integrates multi-scale CLIP embeddings with confidence-aware fusion and a dual-memory architecture consisting of Short-Term Memory (STM) and Long-Term Memory (LTM). The STM aggregates noisy visual observations using geometric, semantic, and temporal confidence cues, while confident and coherent cells are promoted to the LTM as persistent semantic landmarks. Designed for deployment with a Jetson Orin-powered UGV alongside SLAM, CrossMaps runs in real time and produces semantic heatmaps that can be queried with natural language to guide rover navigation.

## 1. 核心问题与整体含义（研究动机与背景）

- **问题背景**：自主巡视器（rover）在非结构化环境中需要构建包含语义信息的地图来指导导航。传统SLAM几何地图缺乏语义理解；近年基于CLIP的开放词汇语义建图（如VLMaps）允许自然语言查询，但在移动平台上部署面临观测噪声大、视角依赖、光照变化、动态物体等问题，导致地图不稳定。
- **核心挑战**：如何在部分可观测、传感器有噪声的条件下，构建鲁棒、可查询、可适应变化（新证据）的长期语义地图。
- **论文目标**：提出CrossMaps——一种置信度感知的开放词汇语义建图流水线，通过双记忆架构（短期记忆STM + 长期记忆LTM）和置信度融合机制，提升在线建图的稳定性和导航可靠性。

## 2. 方法论：核心思想与技术细节

- **整体思路**：在VLMaps基础上，引入**置信度感知融合**和**双记忆架构**。STM聚合近期观测并衰减噪声，LTM仅存储经置信度、一致性和多视角验证的可靠地标。
- **关键技术步骤**：
  1. **多尺度CLIP嵌入提取**：将RGB帧划分为粗粒度和细粒度图像块（tiles），用预训练CLIP ViT-L/14编码每个tile，获得语义嵌入。
  2. **空间投影**：利用深度图和SLAM位姿，采样深度点并反投影到3D，与对应tile的融合嵌入关联，分配到俯视网格单元。
  3. **置信度计算**：每个观测点获得**几何置信度**（距离衰减、多视角提升）、**语义置信度**（与当前单元嵌入的余弦相似度，一致则增强，不一致则软抑制）、**时间置信度**（老观测随时间衰减）。
  4. **STM单元融合**：维护累加器 $S_\mathcal{C}$（置信度加权嵌入和）和 $W_\mathcal{C}$（累积置信度）。融合公式：$S_\mathcal{C} \leftarrow S_\mathcal{C} + c_i e_i$, $W_\mathcal{C} \leftarrow \lambda W_\mathcal{C} + c_i$, $e_\mathcal{C} = S_\mathcal{C} / \|S_\mathcal{C}\|$（$\lambda$为时间衰减因子）。
  5. **STM→LTM提升**：满足三个条件（置信度>阈值$\tau_c$、语义一致性>阈值$\tau_h$、多视角多样性）的单元被提升到LTM，成为持久地标。若新高置信观测与LTM冲突，可替换。
  6. **语义查询**：将自然语言查询用CLIP文本编码，计算与每个单元嵌入的余弦相似度，用单元一致性（coherence）加权得到热图，高置信度单元作为导航目标。
